Quick help

Basic commands:

> s <query> - find query in google by default

Categories:

> new category <name> <color> - create a new category

> delete category <name> - delete a category

Items:

> new item <name> <category> <url> - create a new item in a category

> delete item <name> - delte an item

> <itemName> - go to a previously saved item
